# My First Web Development Project 🚀

Tämä on ensimmäinen projektini, jonka tein itse opiskellessani HTML-kieltä maaliskuussa 2026.

## Mitä tässä projektissa on?
* **Oppimispäiväkirja (Diary):** Kirjasin ylös kaiken, mitä opin jokaisena päivänä.
* **HTML5 & CSS3:** Käytin semanttista koodia, taulukoita, lomakkeita ja multimediatekniikoita (video ja audio).
* **Tehtävät:** Harjoitussivuja, kuten "Geralt of Rivia" -opas ja sanakirja-proto.

## Kuinka katsoa projektia?
1. Avaa tiedosto `index.html` selaimessa.
2. Siirry päiväkirjaan (Diary) nähdäksesi etenemiseni.

## Tavoitteeni
Haluan päästä **oppisopimukseen** ja kehittyä ammattilaiseksi web-kehityksessä. Suoritin HTML-peruskurssin vain 11 päivässä ja opiskelen tällä hetkellä CSS-tyylejä.
